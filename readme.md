# ShowPic
ShowPic is a command-line tool to display pictures inside terminal.

## Installation
Inside the project folder:
```
npm i
npm link
```

## Usage
Globally installed:
```
showpic sp somePic.png
```
Using without installation :
```
node showPic.js showpic somePic.png
```
